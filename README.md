# theKitchen

Live website: https://kyumcu.github.io/theKitchen/

This repository hosts a static personal website published with GitHub Pages.

Main sections:

- Landing page: https://kyumcu.github.io/theKitchen/
- Reports: https://kyumcu.github.io/theKitchen/reports/
- Blog: https://kyumcu.github.io/theKitchen/blog/

The site is plain static HTML and is intended to stay easy to read and edit.

## Local preview

Run:

`./scripts/serve.sh`

Then open `http://127.0.0.1:8000/`.

## Formatting

If you have Prettier installed, run:

`./scripts/format.sh`
